# Functional Programming

### About the project

This project is created to accomplish a number of algorithmic tasks focused on
familiarize known functional concepts in the context of JavaScript.

### Goals:
  * Create a library of utilities following the principles of functional programming
  * Ensure reusability, purity, and immutability in utility design
  * Cover common functional programming concepts such as partial application, currying, memoization and so on

### Technology stack:
  * Typescript
  * Webpack
  * Eslint
  * Prettier
  * Husky
  * Jest
